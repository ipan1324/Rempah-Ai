Tentang Proyek

Penerapan Deep Learning untuk mengenali bumbu dapur tradisional Indonesia secara otomatis melalui analisis citra digital.

MobileNetV2

Arsitektur Convolutional Neural Network yang ringan dan efisien, dilatih pada ImageNet (1.4 juta gambar, 1000 kelas). Pretrained weights-nya digunakan sebagai feature extractor melalui teknik Transfer Learning.

ImageNet Weights

Transfer Learning

Seluruh layer base model di-freeze agar fitur generik tetap terjaga. Ditambahkan custom classification head: GlobalAveragePooling → Dropout → Dense(128) → Dropout → Dense(3). Hanya head ini yang dilatih ulang.

Frozen Base Model

Dataset Bumbu

Dataset berisi citra bumbu dapur Indonesia dari Kaggle (Spices Classification) yang terdiri dari 3 kelas: Kencur, Kunyit, dan Lengkuas. Gambar diproses menjadi ukuran 224×224 piksel.
